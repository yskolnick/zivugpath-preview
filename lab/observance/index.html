What this is: six ways to render the observance/hashkafa field on the shared profile, the same person each time. Switch the Example person up top. Approach 0 is the current linear meter (the one we’re questioning); D is the recommended replacement.

Why this exists: research (2026-07-08) found hashkafa is multidimensional and its categories aren’t linearly ordered; real frum products (JWed, SawYouAtSinai) use categorical labels + specifics, not sliders. See the observance memo.
