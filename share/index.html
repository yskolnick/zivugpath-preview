The link, in a chat

Paste the link — it unfurls into a card.

When a shadchan drops the link into WhatsApp, it opens into a designed preview: her photo, her name, and one warm line. Glanceable like an image, but it's a live link — so it's never a stale screenshot.

  • Her photo stays gated. The preview shows a tasteful portrait; the fuller gallery still opens only on the full link.
  • Always current. Update the profile once and every card already sent reflects it.
  • One tap to the real thing. The card is the hook; the full editorial profile is a tap away.

The one-pager

Prefer to send a picture? Export the one-pager.

Some people forward an image, not a link — a single clean sheet with the essentials in bullets. ZivugPath builds one from the same profile: photo, a few facts, her words in her voice, references on request. Save it as an image or PDF and send it however you like.

  • Same content, bullet voice. The prose profile and the one-pager come from one source — write once.
  • Screenshot-proof. It's a real export, laid out to read well the moment it lands in a thread.
  • Links back to the live page. The footer carries the link (and a QR) so anyone can reach the always-current version.
